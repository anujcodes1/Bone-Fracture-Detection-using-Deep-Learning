# FractureAI

Final year project structure for bone fracture detection using Flask.

## Project Layout

- `backend/`: Flask application code and routes.
- `model/`: Trained model files, checkpoints, and inference utilities.
- `dataset/`: Training, validation, and test data.
- `uploads/`: User-uploaded X-ray images for prediction.
- `outputs/gradcam/`: Generated Grad-CAM heatmaps.
- `reports/pdf/`: Exported PDF reports.
- `templates/`: HTML templates for Flask pages.
- `static/css/`: Stylesheets for the frontend.
